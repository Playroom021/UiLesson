# UiLesson
gethereing all ui components 


UI components are reusable interface elements used to build websites and apps. Common types include:

* **Navigation:** Navbar, sidebar, tabs, breadcrumbs, pagination, menus, bottom navigation.
* **Buttons & actions:** Buttons, icon buttons, floating action buttons (FAB), button groups, dropdown buttons.
* **Form & input:** Text fields, textareas, checkboxes, radio buttons, switches/toggles, sliders, date pickers, file uploads, search bars, select/dropdowns, OTP inputs.
* **Content display:** Cards, lists, tables, accordions, avatars, badges, chips/tags, tooltips, carousels, timelines, calendars.
* **Feedback & status:** Alerts, toast notifications, progress bars, spinners/loaders, skeleton loaders, success/error messages.
* **Overlays:** Modals, dialogs, drawers, popovers, bottom sheets, context menus.
* **Data visualization:** Bar charts, line charts, pie/donut charts, gauges, heatmaps, dashboards.
* **Media:** Image galleries, video players, audio players, image uploaders.
* **Layout:** Containers, grids, stacks, dividers, spacers, resizable panels.
* **Advanced components:** Command palettes, rich-text editors, drag-and-drop areas, Kanban boards, data grids, tree views, steppers, chat interfaces.

A useful way to learn UI design is to start with **buttons → forms → cards → navigation → modals → tables → complete page layouts**.
